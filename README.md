# ATM-project-py
 🏦 ATM Management System (Python)

📌 Overview

This is a simple ATM Management System built using Python.
It simulates basic banking operations like checking balance, depositing money, withdrawing money, and viewing transaction statements.

---

🚀 Features

- 🔐 Basic menu-driven system
- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 📄 View Transaction Statement
- 🔁 Runs in an infinite loop until user exits

---

🛠️ Tech Stack

- Python (Core Concepts)
- Functions & Modular Programming
